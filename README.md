# Gravity-Gomoku
# 一、项目简介
本项目为数算课程大作业，基于 Python + Pygame 实现重力下落式二维五子棋，内置双人对战、AI 人机对战双模式；其中 AI 采用 Minimax 算法 + Alpha-Beta 剪枝优化，区分活棋 / 眠棋，攻防权重平衡，根据几种不同的连棋情况进行局面估值，分为简单 / 中等 / 困难三档推演深度；完整实现界面导航、难度选择、悔棋、重置、胜负判定、独立 exe 打包等全套功能。

# 二、技术栈

    编程语言：Python 3.8+
    图形库：Pygame 2.5
    AI 核心算法：Minimax 极大极小搜索、Alpha-Beta 剪枝、自定义局面估值函数
    打包工具：PyInstaller

# 三、功能清单
## 1. 多页面导航系统

首页：三个功能按钮
    TWO PLAYERS：双人本地对战
    VS AI PLAYER：进入 AI 难度选择页
    HOW TO PLAY：游戏规则说明页
AI 难度选择页
    三档难度：EASY（推演深度 3）、MEDIUM（深度 4）、HARD（深度 5）
    切换落子先后手：人类先手 / AI 先手
    右上角 BACK 返回首页
对局棋盘页面
    右上角 BACK 一键返回首页
    左上角实时提示当前操作玩家
    胜利 / 平局文字提示

## 2. 游戏核心规则
重力落子机制：点击列，棋子下落至列最低空位
棋子配色：玩家 1 黑色，玩家 2/AI 白色，深色背景对比度高
获胜判定：同层横 / 竖 / 斜向连续 5 颗同色棋子获胜；棋盘填满无五子则平局
操作快捷键
    R：清空棋盘，重新开局
    U：悔棋
        双人模式：仅撤销上一步玩家落子
        人机模式：一次性撤回人类 + AI 两步，回到人类落子前状态
本部分内容在游戏界面中的 HOW TO PLAY 也有详细介绍

## 3. AI 智能模块（课程算法核心）

三层难度梯度，对应不同 Minimax 搜索深度，算力与强度平衡
Alpha-Beta 剪枝：大幅剔除无意义分支，降低搜索耗时
强化攻防型局面估值函数
    区分活四、眠四、活三、眠三、活二，不同连线会赋予差异化分值
    防守权重略高于普通进攻：对手活四、活三优先级高于自身普通连线，AI 主动封堵威胁
    棋盘中心区域额外加分，AI 优先抢占核心点位
落子逻辑：人类落子后 AI 自动计算最优位置并重力落子

## 4. 打包部署
支持 PyInstaller 一键打包为独立 Windows exe，无控制台黑窗口，可直接分发运行。

### 环境部署 & 运行教程
方式 1：Python 源码直接运行

安装依赖库

    bash
    pip install pygame pyinstaller

运行程序

    bash
    python Gravity_Gomoku.py

方式 2：打包为独立 exe（Windows）
在代码所在文件夹打开 CMD，执行打包命令：

    bash
    pyinstaller -F -w Gravity_Gomoku.py

打包完成后进入 dist 文件夹，双击 Gravity_Gomoku.exe 即可运行，无需安装 Python 环境。

操作说明

    首页选择游玩模式，AI 模式需先选择难度
    对局中鼠标点击列完成落子，棋子自动下落
    快捷键
        U：悔棋
        R：重置整局棋盘
    右上角 BACK 按钮随时返回主菜单

    Python 版本：3.8 及以上
    内存：≥ 2GB
